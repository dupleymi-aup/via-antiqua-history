## О проекте

**История Древнего Пути** — комплексная интерактивная историческая энциклопедия античного мира, охватывающая четыре великие цивилизации: Древнюю Грецию, Римскую империю, Месопотамию и Кубань (античное Причерноморье). Проект представляет эти цивилизации как части единого культурно-экономического пространства, связанного торговыми путями, языком и культурным обменом.

Платформа объединяет:
- **4 цивилизации** с детальным описанием
- **18 городов** — от Афин и Рима до Ура и Пантикапея
- **32+ памятника архитектуры** с подробными описаниями
- **12 исторических персоналий** с биографиями
- **7 чудес света** — все с описаниями и наследием
- **Интерактивную ленту времени** — 30+ событий от 3500 до н. э. до 1453 н. э.
- **Интерактивную карту** с фильтрами по регионам
- **Авторский исторический анализ** — 5 эссе о связях цивилизаций
- **Глоссарий** из 22 терминов с поиском
- **Квиз** на 20 вопросов для проверки знаний

Все материалы опираются на современные исторические исследования, включая труды по античной истории, археологии и сравнительному цивилиноведению.

## Система авторского анализа

Авторский исторический анализ раскрывает связи между цивилизациями через пять ключевых эссе:

1. **Эллинизм как культурный мост** — как греческая культура стала основой для взаимодействия Рима, Египта и Ближнего Востока
2. **Pax Romana и экономическая интеграция** — как римский мир создал единую экономическую зону от Британии до Месопотамии
3. **Торговые пути античности** — морские и сухопутные маршруты, связывавшие цивилизации
4. **Культурный синтез в Причерноморье** — как Кубань стала местом взаимодействия греков, варваров и римлян
5. **Наследие античности** — влияние древних цивилизаций на современный мир

## Установка и запуск

### Предварительные требования

- **Node.js** версии 18.17 или выше (рекомендуется 20+)
- **npm** (или **yarn**, **pnpm**)

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/QuadDarv1ne/via-antiqua-history.git
cd via-antiqua-history

# Установить зависимости
npm install

# Скопировать файл окружения
cp .env.example .env.local

# Запустить в режиме разработки
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

### Сборка для продакшена

```bash
# Сборка проекта
npm run build

# Запуск собранного приложения
npm start
```

### Линтинг

```bash
# Проверка кода
npm run lint
```

## Интерактивность

### Навигация и поиск
- **Глобальный поиск** (`Ctrl+K` / `Cmd+K` / `/`) — мгновенный поиск по городам, памятникам, терминам и персоналиям
- **Sticky-навигация** — фиксированное меню с smooth scroll к секциям
- **Закладки** — сохранение любимых материалов в localStorage, плавающая кнопка для быстрого доступа
- **Reading progress** — визуальный индикатор прогресса чтения вверху экрана
- **Кнопка «наверх»** — появляется при скролле вниз

### Интерактивные элементы
- **Табы** — переключение городов внутри региона
- **Модальные диалоги** — подробные карточки достопримечательностей, персоналий и чудес света
- **Интерактивная карта** — кликабельные маркеры с фильтрами по 4 регионам
- **Лента времени** — навигация стрелками `←`/`→`, анимация при скролле
- **Сравнительная таблица** — 8 критериев для каждой цивилизации
- **Глоссарий** — живой поиск с фильтрацией
- **Квиз** — навигация клавишами `1`–`4` для ответов, `←`/`→` для переходов

### Темы и адаптивность
- **Тёмная/светлая тема** — античный стиль с пергаментным фоном и бронзовыми акцентами
- **Полная адаптивность** — mobile-first подход, корректное отображение на всех устройствах
- **Анимации** — плавные переходы и появление секций при скролле (Framer Motion)
