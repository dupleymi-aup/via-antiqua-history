## About the Project

**History of the Ancient Way** is a comprehensive interactive historical encyclopedia of the ancient world, covering four great civilizations: Ancient Greece, the Roman Empire, Mesopotamia, and Kuban (ancient Black Sea region). The project presents these civilizations as parts of a single cultural and economic space connected by trade routes, language, and cultural exchange.

The platform combines:
- **4 civilizations** with detailed descriptions
- **18 cities** — from Athens and Rome to Ur and Panticapaeum
- **32+ architectural monuments** with detailed descriptions
- **12 historical personalities** with biographies
- **7 Wonders of the World** — all with descriptions and legacy
- **Interactive timeline** — 30+ events from 3500 BCE to 1453 CE
- **Interactive map** with region filters
- **Author's historical analysis** — 5 essays on civilization connections
- **Glossary** of 22 terms with search
- **Quiz** with 20 questions for knowledge testing

All materials are based on modern historical research, including works on ancient history, archaeology, and comparative civilizational studies.

## Author's Analysis System

The author's historical analysis reveals connections between civilizations through five key essays:

1. **Hellenism as a Cultural Bridge** — how Greek culture became the foundation for interaction between Rome, Egypt, and the Near East
2. **Pax Romana and Economic Integration** — how the Roman peace created a unified economic zone from Britain to Mesopotamia
3. **Ancient Trade Routes** — maritime and land routes connecting civilizations
4. **Cultural Synthesis in the Black Sea Region** — how Kuban became a place of interaction between Greeks, barbarians, and Romans
5. **Legacy of Antiquity** — influence of ancient civilizations on the modern world

## Installation and Setup

### Prerequisites

- **Node.js** version 18.17 or higher (20+ recommended)
- **npm** (or **yarn**, **pnpm**)

### Installation

```bash
# Clone the repository
git clone https://github.com/QuadDarv1ne/via-antiqua-history.git
cd via-antiqua-history

# Install dependencies
npm install

# Copy environment file
cp .env.example .env.local

# Run in development mode
npm run dev
```

The application will be available at [http://localhost:3000](http://localhost:3000)

### Production Build

```bash
# Build the project
npm run build

# Run the built application
npm start
```

### Linting

```bash
# Check code
npm run lint
```

## Interactivity

### Navigation and Search
- **Global search** (`Ctrl+K` / `Cmd+K` / `/`) — instant search across cities, monuments, terms, and personalities
- **Sticky navigation** — fixed menu with smooth scroll to sections
- **Bookmarks** — save favorite materials to localStorage, floating button for quick access
- **Reading progress** — visual reading progress indicator at the top of the screen
- **Scroll to top button** — appears when scrolling down

### Interactive Elements
- **Tabs** — switch between cities within a region
- **Modal dialogs** — detailed cards for monuments, personalities, and Wonders
- **Interactive map** — clickable markers with filters by 4 regions
- **Timeline** — navigation with `←`/`→` arrows, animation on scroll
- **Comparative table** — 8 criteria for each civilization
- **Glossary** — live search with filtering
- **Quiz** — keyboard navigation with `1`–`4` for answers, `←`/`→` for transitions

### Themes and Responsiveness
- **Dark/Light theme** — antique style with parchment background and bronze accents
- **Full responsiveness** — mobile-first approach, correct display on all devices
- **Animations** — smooth transitions and section appearance on scroll (Framer Motion)
