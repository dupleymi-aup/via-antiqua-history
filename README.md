## О проекте

**История Древнего Пути** — интерактивная историческая энциклопедия античного мира, охватывающая Древнюю Грецию, Римскую империю, Месопотамию и Кубань как части единого культурно-экономического пространства. Платформа объединяет **4 цивилизации**, **18 городов**, **32+ памятника**, **30+ персоналий**, **7 чудес света**, интерактивную ленту времени, карту, сравнительные таблицы, авторский исторический анализ, глоссарий из **22 терминов** и квиз на **20 вопросов**.

**Полные описания доступны на русском и английском языках:**

- [README на русском языке](README_RU.md)
- [README in English](README_EN.md)

### Ключевые возможности

- **4 цивилизации** — Древняя Греция, Римская империя, Месопотамия, Кубань
- **18 городов** — от Афин и Рима до Ура и Пантикапея
- **32+ памятника** — Парфенон, Колизей, Висячие сады, Херсонес и др.
- **12 персоналий** — Перикл, Александр Македонский, Цезарь, Ганнибал и др.
- **7 чудес света** — все с описаниями и наследием
- **8 архитектурных ордеров** — SVG-схемы капителей
- **8 исторических эпох** — от крито-микенской до поздней античности
- **Интерактивная лента времени** — 30+ событий, 3500 до н. э. — 1453 н. э.
- **Интерактивная карта** — 18 городов с фильтрами по 4 регионам
- **Сравнительная таблица** — 8 критериев × 4 цивилизации
- **Авторский анализ** — 5 эссе о торговых путях, эллинизме, Pax Romana и др.
- **Глоссарий** — 22 термина с поиском и фильтрацией
- **Квиз** — 20 вопросов с итоговой статистикой
- **Глобальный поиск** — `Ctrl+K` / `Cmd+K` для быстрого доступа
- **Закладки** — сохранение в localStorage
- **Тёмная/светлая тема** — античный стиль с пергаментным фоном

### Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/QuadDarv1ne/via-antiqua-history.git
cd via-antiqua-history

# Установить зависимости
bun install

# Запустить в режиме разработки
bun run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)
