# История Древнего Пути (Исторический Лабиринт)

**Образовательный интерактивный ресурс о цивилизациях Древней Греции, Рима, Месопотамии и Кубани**

**Автор проекта:** Дуплей Максим Игоревич

## О проекте

«История Древнего Пути» — интерактивная историческая энциклопедия античного мира. Представляет Древнюю Грецию, Римскую империю, Месопотамию и Кубань как части единого культурно-экономического пространства. Помимо описаний городов и памятников, содержит авторский исторический анализ связей этих территорий через торговые пути, язык и культуру.

Стартовая точка — статья «Прогулка по афинскому Акрополю» из «Яндекс.Путешествий».

```bash

bun run dev          # dev-сервер → http://localhost:3000
```

## Интерактивность

- **Глобальный поиск** (`Ctrl+K` / `Cmd+K` / `/`) — города, памятники, термины, персоналии
- **Закладки** — сохраняются в localStorage, плавающая кнопка для доступа
- **Reading progress** — полоска прогресса скролла вверху экрана
- **Keyboard nav в квизе** — `1`–`4` для ответов, `←`/`→` для навигации
- **Keyboard nav в таймлайне** — стрелки `←`/`→` для перехода между событиями
- Sticky-навигация с smooth scroll
- Табы для переключения городов внутри региона
- Модальные диалоги для достопримечательностей, персоналий и чудес
- Кликабельная карта с фильтрами
- Глоссарий с живым поиском
- Тёмная/светлая тема с античным стилем
- Кнопка «наверх» при скролле
- Полная адаптивность (mobile-first)
- Анимации появления секций при скролле

## Дизайн

- **Палитра:** пергаментный фон, бронзовый primary, терракотовый акцент
- **Цвета регионов:**
  - Греция — золото `oklch(0.55 0.13 70)`
  - Рим — терракота `oklch(0.55 0.13 35)`
  - Месопотамия — охра `oklch(0.55 0.13 50)`
  - Кубань — изумруд `oklch(0.5 0.11 145)`
- Декоративный меандр-бордюр (античная керамика)
- Тёмная тема: глубокий бронзовый фон с медным акцентом
